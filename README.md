# Lazy Image

Camp Jefferson Lazy Image utility, in the vein of Gatsby Image

## Usage

### Installation

```
npm install @campj/lazy-image --save-dev
```

### Import into js file for client

```js
import { LazyImage } from "@campj/lazy-image";
```

#### Initialize

```js
LazyImage.init();
```

#### Re-run (dom elements changed)

```js
LazyImage.run();
```
